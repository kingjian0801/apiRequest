# apiHttp
## api请求模拟器
这是一个模拟远程API访问的脚本，整个脚本就一个html文件，用浏览器打开就可用。
### 主要实现以下功能
- [x] 请求方式支持：GET,POST,PUT,DELETE,HEAD,OPTIONS,PATCH
- [x] 请求参数写入 or 删除
- [ ] 请求参数支持json字符串
- [ ] 显示http响应头信息
- [x] 显示请求结果

==========================================================
#### 目前只能模拟 GET, POST访问，并显示json数据或html数据，Http头信息F12看好了，更详细，后面再优化
